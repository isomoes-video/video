中文标题：iChat 在线聊天站点介绍
English Title: Introducing the iChat Web Chat Platform

这期视频围绕 iChat 这个网页在线聊天端展开，介绍它为什么会被做出来，以及它和现有 iAPI 体系之间的关系。视频先说明了一个现实问题：API 站点虽然适合接入 OpenCode、Claude Code 等 Agent 工具，但对很多普通用户来说，直接通过 API 聊天的门槛仍然偏高。因此，作者基于已有开源项目进行了二次开发，做出了一个更适合直接访问和使用的大模型聊天网页，让用户可以用更低成本的方式体验模型能力。

随后，视频重点说明了 iChat 的产品能力和系统整合方式。它与 iAPI 的用户体系、登录体系和支付体系打通，用户可以直接使用 iAPI 账号登录、查看余额、充值 Token，并同步使用上游模型列表。与此同时，iChat 也同步了注册逻辑，尽量减少用户必须往返后台的操作。视频还强调，这个站点的一大价值在于，用户无需复杂配置，便可以直接访问平台提供的大模型服务。

在功能层面，作者展示了 iChat 不只是一个普通聊天页面，还支持普通聊天、Web Search 和 Deep Research 等模式。结合其前端加 Rust 后端的架构，以及提示词注入、自定义工具、网页抓取参数、搜索引擎等能力，这个在线聊天端已经具备了一个轻量化 Agent 的雏形，可以处理更复杂、更长链路的任务。视频最后介绍了项目的开源状态、当前仍在修复的文件上传问题，以及团队后续继续扩展 Agent 能力、在本地或沙盒环境中进一步释放能力的规划。
