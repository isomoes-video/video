中文标题：Isomoes API 接入 Claude Code 教程
English Title: Connecting Claude Code to Isomoes API

本视频介绍如何将 Claude Code 接入 Isomoes API（aiapi）平台，以使用平台上提供的 Claude 及其他模型。主讲人以 macOS/Linux 安装方式为例，演示了从安装 Claude Code 到配置 API 密钥、选择模型、验证调用的完整流程。

安装方面，需先安装 Node.js，再通过 npm 安装 Claude Code。配置方面，可临时在终端中设置环境变量（复制平台提供的 API URL 和 Key），也可将配置持久化写入 `.claude/settings.json`，其中包含 API 站点地址和模型 ID。由于 Isomoes API 同时支持 Anthropic 协议和 OpenAI/GPT 协议，因此 Claude Code 下也可调用 GPT 系列模型。

配置完成后，主讲人启动 Claude Code，让其查询 GPU 版本信息和 CPU 信息，验证模型可正常响应。通过平台使用日志可确认调用记录，首字时间在十秒以内，体验与官方接口相近，证明 Claude Code 可无缝集成 Isomoes API。
